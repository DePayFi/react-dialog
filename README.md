## Quickstart

```
yarn add depay-react-dialog
```

or 

```
npm install --save depay-react-dialog
```

```javascript
import { renderDialog } from ReactDialog;

renderDialog({
  container: document,
  content: React.createElement('h1', {}, 'I am a dialog!')
});
```

## Development

### Get started

```
yarn install
yarn start
```

### Release

```
npm publish
```
