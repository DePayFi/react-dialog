## Quickstart

```
yarn add depay-react-dialog
```

or 

```
npm install --save depay-react-dialog
```

```javascript
import ReactDialog from 'depay-react-dialog';

render() {
  return(
    ...
    <ReactDialog>
      <h1>I am a dialog</h1>
    </ReactDialog>
  )
}
```

## Development

### Get started

```
yarn install
yarn start
```

### Release

```
npm publish
```
